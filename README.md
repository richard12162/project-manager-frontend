# Project Manager Frontend

React + TypeScript + Vite Frontend fuer ein Java/Spring Task-Management-Backend.

Aktueller Stand:

- Basis-Routing fuer Auth und Projects
- Minimale App-Shell fuer den produktiven Einstieg
- Vorhandene OpenAPI-Typen in `src/types/api.ts`

Entwicklung:

```bash
npm install
npm run dev
```
